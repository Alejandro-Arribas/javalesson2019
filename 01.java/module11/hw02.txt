
Задание №1

В продолжении ранее выданного дз о базе данных налоговой испекции по штрафам:

		Реализовать базу данных налоговой инспекции по штрафам. Идентифицировать конкретного человека бу-
		дет его персональный идентификационный код. У одного человека может быть много штрафов.
		Реализовать: 
		1. Полная распечатка базы данных.
		2. Распечатка данных по конкретному коду.
		3. Распечатка данных по конкретному типу штрафа.
		4. Распечатка данных по конкретному городу.
		5. Добавление нового человека с информацией о нем.
		6. Добавление новых штрафов для уже существующей записи.
		7. Удаление штрафа.
		8. Замена информации о человеке и его штрафах.
		
Реализовать возможность сохранения/считывания информации в файл в текстовом виде

Задание №2

Написать приложение для управлением персональным списком задач (ToDoList)
Реализовать функционал:

1. Регистрация нового пользователя
2. Авторизация пользователя
3. Отображение списка всех не выполненых задач (отсортированные по важности и дате)
4. Отображение списка всех выполненных задач (отсортированные по дате в обратном порядке)
5. Добавление новой задачи
6. Отметка про выполнение задания
7. Удаление задачи
8. Архивирование выполненных задач (после архивирования исчезают из списка)
9. Выход

Каждая задача должна включать следующие данные:
1. Название
2. Важность (от 1 до 5)
3. Срок выполнения (опционально)
4. Категория (опционально)
